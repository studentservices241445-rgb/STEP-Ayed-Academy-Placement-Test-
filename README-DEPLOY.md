# طريقة رفع الموقع بسرعة (بدون برمجة)

## Netlify (الأبسط)
1) ادخل https://app.netlify.com/drop  
2) اسحب المجلد كامل وارمه في الصفحة  
3) خلال ثواني بيطلع لك رابط الموقع  
4) لو تبغى اسم نطاق (Domain) تقدر تضيفه من إعدادات الموقع

## GitHub Pages
1) أنشئ Repository جديد  
2) ارفع كل ملفات المشروع  
3) Settings → Pages → Deploy from branch → main → /root  
4) بتطلع لك صفحة موقع رسمي

## ملاحظة مهمة
إذا غيّرت أي ملفات بعد الرفع:
- ارفع التحديثات مرة ثانية
- أو اعمل Commit جديد (في GitHub)

## ملاحظة عن PWA
التثبيت يشتغل تلقائياً على المتصفحات اللي تدعمه.  
على iPhone: Safari → Share → Add to Home Screen
